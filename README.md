# QuantityMeasurementApp-Frontend# 🚀 Quantity Measurement Application – Frontend (Angular)

## 📏 Scalable & Responsive UI for Quantity Conversion and Measurement Operations

A modern frontend built using **Angular** and **TypeScript** for performing unit conversion, unit comparison, and arithmetic operations on measurable quantities.

---

## 📌 Project Overview

The **Quantity Measurement Application – Angular Frontend** provides a clean and responsive UI for handling quantity-based operations.

This implementation focuses on:

- Component-based architecture  
- Reusable UI modules  
- Service-driven logic  
- Reactive programming  
- Clean and maintainable structure  

---

## 🚀 How to Run

```bash
git clone https://github.com/mohd-meraaz/QuantityMeasurementApp-Frontend
cd <project-folder>
npm install
ng serve
```

Open in browser:

```
http://localhost:4200
```
